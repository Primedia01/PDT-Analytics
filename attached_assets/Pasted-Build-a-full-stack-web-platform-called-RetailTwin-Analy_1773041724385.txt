Build a full-stack web platform called RetailTwin Analytics.

The platform will allow users to visualize digital twins of shopping malls, overlay digital advertising assets, and display audience analytics around those assets across a portfolio of malls.

The system must be scalable but initially built as an MVP supporting 27 shopping malls.

Use modern, scalable technologies.

Recommended stack:

Frontend

React

Three.js (for 3D digital twins)

TailwindCSS

React Three Fiber

Backend

Node.js + Express

REST API

Database

PostgreSQL

Analytics

Chart.js or Recharts

Deployment

Replit Deployments

CORE PLATFORM FUNCTION

The system should allow users to:

View a portfolio dashboard

Select a shopping mall

Load the 3D digital twin of the mall

See advertising assets placed in the mall

Click an asset to see audience analytics

See portfolio-level analytics across all malls
